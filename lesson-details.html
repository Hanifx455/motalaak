محتوى الدرس

مقدمة في المعادلات من الدرجة الثانية

المعادلة من الدرجة الثانية هي معادلة يمكن كتابتها على الصورة: ax² + bx + c = 0، حيث a، b، c أعداد حقيقية و a ≠ 0. في هذا الدرس، سنتعلم كيفية حل هذه المعادلات باستخدام عدة طرق مختلفة.

الصيغة العامة للمعادلة من الدرجة الثانية

الصيغة العامة للمعادلة من الدرجة الثانية هي:

ax² + bx + c = 0

حيث:

  • a هو معامل x²، ويجب أن يكون غير صفري (a ≠ 0)
  • b هو معامل x
  • c هو الحد الثابت

طرق حل المعادلات من الدرجة الثانية

1. القانون العام (الصيغة العامة)

يمكننا استخدام القانون العام لحل أي معادلة من الدرجة الثانية:

x = (-b ± √(b² - 4ac)) / (2a)

حيث:

  • b² - 4ac يسمى المميز (Δ)
  • إذا كان Δ > 0، فإن المعادلة لها حلان حقيقيان مختلفان
  • إذا كان Δ = 0، فإن المعادلة لها حل حقيقي واحد (جذر مضاعف)
  • إذا كان Δ < 0، فإن المعادلة ليس لها حلول حقيقية (الحلول مركبة)

2. التحليل إلى العوامل

إذا كان من الممكن تحليل الطرف الأيسر من المعادلة إلى عوامل، فيمكننا كتابة المعادلة على الصورة:

(x - r)(x - s) = 0

وبالتالي، فإن الحلول هي x = r و x = s.

3. إكمال المربع

طريقة إكمال المربع هي تقنية مفيدة لتحويل المعادلة من الدرجة الثانية إلى صيغة مربع كامل:

  1. نقسم المعادلة على a (إذا كان a ≠ 1)
  2. ننقل الحد الثابت إلى الطرف الأيمن
  3. نضيف (b/2a)² إلى كلا الطرفين
  4. نعيد كتابة الطرف الأيسر كمربع كامل
  5. نحل المعادلة

أمثلة تطبيقية

مثال 1: حل المعادلة x² - 5x + 6 = 0

باستخدام القانون العام:

a = 1, b = -5, c = 6

Δ = b² - 4ac = (-5)² - 4(1)(6) = 25 - 24 = 1

x = (-b ± √Δ) / (2a) = (5 ± √1) / (2) = (5 ± 1) / 2

x₁ = (5 + 1) / 2 = 3

x₂ = (5 - 1) / 2 = 2

إذن، الحلول هي x = 2 و x = 3.

باستخدام التحليل إلى العوامل:

x² - 5x + 6 = (x - 2)(x - 3) = 0

إذن، x = 2 أو x = 3.

مثال 2: حل المعادلة 2x² + 4x - 6 = 0

باستخدام القانون العام:

a = 2, b = 4, c = -6

Δ = b² - 4ac = 4² - 4(2)(-6) = 16 + 48 = 64

x = (-b ± √Δ) / (2a) = (-4 ± √64) / (2(2)) = (-4 ± 8) / 4

x₁ = (-4 + 8) / 4 = 4/4 = 1

x₂ = (-4 - 8) / 4 = -12/4 = -3

إذن، الحلول هي x = 1 و x = -3.

مثال 3: حل المعادلة 3x² - 6x + 3 = 0

باستخدام القانون العام:

a = 3, b = -6, c = 3

Δ = b² - 4ac = (-6)² - 4(3)(3) = 36 - 36 = 0

بما أن Δ = 0، فإن المعادلة لها حل واحد (جذر مضاعف):

x = -b / (2a) = 6 / (2(3)) = 6/6 = 1

إذن، الحل هو x = 1 (جذر مضاعف).

تطبيقات عملية

المعادلات من الدرجة الثانية لها العديد من التطبيقات في الحياة اليومية والعلوم المختلفة:

  • حساب المسافة والسرعة والزمن في الفيزياء
  • حساب المساحة والحجم في الهندسة
  • تحليل البيانات في الإحصاء
  • حل مسائل التكلفة والربح في الاقتصاد

خلاصة

في هذا الدرس، تعلمنا كيفية حل المعادلات من الدرجة الثانية باستخدام عدة طرق:

  1. القانون العام (الصيغة العامة)
  2. التحليل إلى العوامل
  3. إكمال المربع

كما تعرفنا على أهمية المميز (Δ) في تحديد عدد الحلول ونوعها.

      منذ يومين
      أهلاً سارة، نعم يمكن حل المثال الثالث باستخدام طريقة إكمال المربع. سنبدأ بقسمة المعادلة على 3 لنحصل على: x² - 2x + 1 = 0. ثم نعيد ترتيبها: x² - 2x = -1. لإكمال المربع، نضيف (b/2)² = (-2/2)² = 1 إلى كلا الطرفين: x² - 2x + 1 = -1 + 1. وهذا يعطينا: (x - 1)² = 0، ومنه x = 1. وهو نفس الحل الذي حصلنا عليه باستخدام القانون العام.
